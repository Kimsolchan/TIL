# 변수의선언과저장연습2
```java
public class Ex2_02_변수의선언과저장연습2 {
	public static void main(String[] args) {
		System.out.println(2+4);
		System.out.println(2-4);
		System.out.println(2*4);
		System.out.println(2/4);
		
        
		int x = 20 , y = 40;
        
		System.out.println(x+y);
		System.out.println(x-y);
		System.out.println(x*y);
		System.out.println(x/y);
	}
}

// result
0
60
-20
800
0
```
