# 변수의타입연습
```java
public class Ex2_03_변수의타입연습 {
	public static void main(String[] args) {
		int number = 60;
        
		double height = 172.55;
        
		char ch = 'a';
        
		String str = "문자열";
		
		System.out.println("number : " + number);
		System.out.println("height : " + height);
		System.out.println("ch : " + ch);
		System.out.println("str : " + str);
	}
}

// result
number : 60
height : 172.55
ch : a 
str : 문자열
```
